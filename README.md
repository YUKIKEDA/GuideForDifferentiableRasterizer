# 微分可能ラスタライザ 完全学習ガイド

初学者から **nvdiffrast** 相当のライブラリをスクラッチで自作できるレベルまでをカバーする、微分可能ラスタライザの包括的教材です。

## 概要

3D ビジョンにおける逆問題（逆レンダリング・形状最適化・テクスチャ推定など）では、レンダリング処理を微分可能にし、勾配に基づいてパラメータを最適化する必要があります。本ガイドでは、その中核となる**微分可能ラスタライゼーション**の理論・数学・実装を、前提知識から GPU 実装・実践応用まで一貫して学べるように構成しています。

## 目次・構成

**メインの目次は [index.md](index.md) を参照してください。**

| Part | 内容 |
|------|------|
| **Part I** | 前提知識（数学・CG・深層学習と自動微分） |
| **Part II** | なぜ微分可能ラスタライザか（逆問題・微分可能性の障壁） |
| **Part III** | ラスタライゼーションの数学（三角形・重心座標・アンチエイリアシング・テクスチャ） |
| **Part IV** | 微分可能ラスタライゼーションの理論（古典的アプローチ・ソフトラスタ・解析的勾配） |
| **Part V** | 実装編 — スクラッチで作る（CPU/GPU・最小構成から本格パイプラインまで） |
| **Part VI** | nvdiffrast レベルへ（アーキテクチャ・性能・堅牢性） |
| **Part VII** | 実践的応用例（逆レンダリング・最適化・座標系の一貫性） |
| **Part VIII** | 演習とプロジェクト・参考文献 |

## 想定読者と到達目標

| 段階 | 読者像 | 到達目標 |
|------|--------|----------|
| Part I–II | 初学者 | なぜ微分可能ラスタライザが必要か説明でき、パイプラインと微分の障壁を理解する |
| Part III–IV | 中級者 | ラスタライゼーションとその微分を数式で扱い、ソフト／解析的アプローチを実装できる |
| Part V–VI | 上級者 | GPU 実装を含む自作ライブラリを設計し、nvdiffrast と同等の機能・品質を目指せる |
| Part VII | 実務志向 | テクスチャ・形状・照明の逆最適化など、実践的応用に繋げられる |
| Part VIII | 全員 | 演習で手を動かし、デバッグ演習を含めて逆レンダリングに繋げられる |
